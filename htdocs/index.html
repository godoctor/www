The Go Doctor is an automated refactoring tool for the Go programming language. It currently supports:

  • Rename
  • Extract Function
  • Toggle var â‡” :=
  • Add Godoc stubs

Try out the Go Doctor using the online demo to the left. Then, download and install it on your machine. There is a Vim plug-in for the Go Doctor, or it can be run directly from the command line.

Can't find your favorite feature? The Go Doctor is open source, so fork it on GitHub.
